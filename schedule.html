Weekly Structure

The day-by-day skeleton. Mondays, Wednesdays, Thursdays, and Saturdays are fixed; Friday and Sunday change with the phase as gym makes way for additional running volume.

Fixed Days Across All Phases

Day Phase 1 (Base) Phase 2 (Build) Phase 3 (Peak)
Monday Gym Upper + Core Gym Upper + Core Gym Upper + Core
Tuesday Run Key Run (intervals/tempo) Run Key Run (intervals/tempo) Run Key Run (race-pace)
Wednesday Gym Lower Body Gym Lower Body Gym Lower Body
Thursday Run Easy Run (Zone 2) Run Easy Run (Zone 2) Run Easy Run (Zone 2)
Friday Gym Full Body + Stairmaster Run Quality Run / Incline Treadmill Run Vert Session / Incline Treadmill
Saturday Run Long Run Run Long Run Run Long Run #1 (hard)
Sunday Rest Rest or Easy Run Long Run #2 (fatigued legs)

Key Design Decisions

  • Upper body on Monday (not Wednesday) so legs are fresh for Tuesday's key quality run
  • Gym tapers 3 -> 2 days as running volume ramps — upper body + core stays throughout for muscle retention
  • Back-to-back weekend long runs in peak phase rather than ever-increasing single Saturday runs
  • Amsterdam hill constraint: completely flat; hill work via stairmaster (Phase 1 Fri gym) and incline treadmill (Phase 2-3) — see Running → Hill Work

Progression Rules

  • Max 10% weekly volume increase
  • Down week every 3-4 weeks (~25-30% reduction)
  • 2-week taper before race day
  • Peak week (Wk 11) = 96 km

Averages

  • Base phase: ~50 km/week
  • Build phase: ~65 km/week
  • Peak phase: ~81 km/week (excl. down weeks)
